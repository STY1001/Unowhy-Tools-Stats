# Unowhy Tools Stats
A small API dev in Node.JS to get stats of [Unowhy Tools](https://github.com/STY1001/Unowhy-Tools) (no user data are collected) by using POST request

Collected data:
- Install ID
- Version
- Build
- Selected lang
- Start mode (normal/tray)
- If tray is enabled
- If is a debug version
